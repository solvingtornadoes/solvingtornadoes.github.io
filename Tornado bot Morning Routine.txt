The Solving Tornadoes Morning Routine
1. Wake up the publishing engine
Run this the moment you open the terminal to activate your custom environment:

Bash

source bot_env/bin/activate

2. Find today's target
List out everything waiting in the vault:

Bash

ls -1 ~/research_vault

(Optional: If you are looking for a specific topic, search inside the files):

Bash

grep -il "keyword" ~/research_vault/*.md

3. Stage the draft
Copy your chosen file from the vault to the drafts folder, giving it a clean, lowercase name with underscores instead of spaces:

Bash

cp ~/research_vault/Your_Messy_Draft_Name.md ~/website_drafts/clean_draft_name.md

4. Add the "ID Badge" and Title
Open the staged file in your text editor:

Bash

gedit ~/website_drafts/clean_draft_name.md &

Make sure the very top of the file looks exactly like this before you hit save:

Markdown
---
layout: default
title: "Your Web Browser Tab Title"
---

# Your Massive On-Page Title

(Your scientific text starts here...)
5. Launch to the live site
Close the editor and fire the conduit to let your bot build the site:

Bash

sync_tornado
